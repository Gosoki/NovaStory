## 0. 一句话概括

**在"AI 大纲 → 人编辑 → AI 成稿"的分镜创作流水线上,做组内三条件实验(全自动 C / 大纲编辑 D / 大纲编辑+AI 异议 D+MM),用"ghost-run 机器反事实分布"作主指标,同时回答两个问题:人该不该进入 AI 流水线(干预点),以及 AI 该不该主动推人一把(异议机制)。**

设计满足你的全部约束:不微调模型、被试 ≤30 人、单人 40 分钟内、全程在线、主指标全自动计算、无论结果正负都有可发表的结论("两头堵")。

## 1. 题目(工作名)

- 中文:《大纲干预与 AI 建设性异议对新手 AI 辅助分镜创作中产出同质化与心理所有权的影响》
- 英文:*Outline-Stage Intervention and Constructive AI Dissent in Novice Storyboard Co-Creation: Effects on Output Homogenization and Psychological Ownership*
- 日文(ゼミ用):「アウトライン段階の人的介入とAIによる建設的異議が、初心者の絵コンテ共創における出力同質化と心理的オーナーシップに与える影響」

对比原标题:去掉了"博弈/赋能/困境/机制"四个大词和多模态阶段规划,小而可守。机制名 **ModeMirror(众数之镜)** 保留为系统贡献的命名。

## 2. 研究问题与假设

### RQ(全部可证伪)

- **RQ1(干预点)**:大纲阶段的人类编辑(D)相比一键全自动(C),能否使产出显著偏离同一模型在相同输入下的纯采样分布,并提升心理所有权?
- **RQ2(异议机制)**:在 D 之上加入 ModeMirror 异议模块(D+MM),能否进一步提升产出的去典型化与多样性,而不损害质量与所有权?
- **RQ3(剂量-反应)**:干预的客观投入量(大纲编辑距离、异议裁决行为)是否连续预测去典型化程度与所有权?二者同步还是解离(只涨所有权不涨多样性 = "控制感幻觉")?

### 假设

| # | 内容 | 性质 |
|---|------|------|
| H1 | 心理所有权:D+MM ≥ D > C(组内配对) | 操纵检验/已知效应复制锚点(Joshi & Vogel 2025、He 2026 已证投入量↔所有权;不成立=操纵失败) |
| H2a | 去典型化(HLZ,见 §4):D > C;H2b:D+MM > D | 主假设 |
| H3 | 集体多样性(格式剥离后 1−组内平均两两余弦):D+MM > D > C,且超出机器采样地板的 bootstrap CI | 主假设(与 H2 联合,封堵"反众数聚类"盲区) |
| H4 | 质量:D 与 C、D+MM 与 D 均等价(预注册 TOST,等价界 ±0.4 dz) | "两头堵"关键——零差异也是可解释结论 |
| H5 | D 内编辑距离、D+MM 内裁决行为(采纳/改造)正向预测 HLZ 与所有权 | 剂量-反应 |
| MC1 | ModeMirror 异议文本确实远离机器众数中心(异议-质心距离 > 用户大纲-质心距离) | 前置操纵检验,不通过则迭代 prompt |
| MC2 | 不同被试收到的异议文本两两相似度不显著高于机器基线相似度 | 防"反众数聚类"操纵检验 |

### 预注册"解释矩阵"(保证好出结果)

| 结果形态 | 结论叙事 | 对话文献 |
|----------|----------|----------|
| HLZ↑ 且所有权 ≥ | 异议机制有效,逆转同质化(主线故事) | Doshi & Hauser; Maier et al. |
| HLZ 不变但所有权 ↑ | "控制感幻觉":编辑入口提升主观所有权却未触动模型默认输出 | AI Ghostwriter Effect; Kumar et al. |
| 异议采纳率极低 | 新手"异议回避"的首份行为证据 | Sarkar (CACM 2024) 立场文的实证回应 |
| HLZ↑ 但所有权 ↓ | 多样性-所有权的帕累托权衡图谱 | 全新发现 |

四个象限均预先写好叙事,任何结果都有论文。

## 3. 实验设计

### 条件(组内,单变量递进)

| 条件 | 流程 | 与上一条件的唯一差异 |
|------|------|---------------------|
| **C(全自动)** | 写 1-2 句意图陈述 → 一键生成终稿 | — |
| **D(大纲干预)** | 意图陈述 → AI 大纲 → 被试编辑大纲 → AI 成稿 | +人类编辑入口 |
| **D+MM(大纲干预+异议)** | 同 D,但在编辑后、成稿前插入 ModeMirror 模块 | +AI 建设性异议 |

**ModeMirror 模块定义**:一次额外 LLM 调用,prompt 注入用户当前大纲 + 3 份预采样的"默认大纲",输出固定三段:① 重合揭示("如果全交给 AI,默认会这样写,你目前与它重合在 X");② 一条挑战性提问;③ 一条远离默认的具体反提案(反提案的发散维度从预定义池——视角/时间结构/基调/类型混搭——按被试随机抽取,防止所有被试被推向同一个"反方向")。UI 强制点击[采纳/改造后采纳/拒绝]之一并可填一句理由,才能生成终稿。

**砍掉的条件**:A(纯手写,地板效应,试运行输出是"1\n2\n3")、B(prompt 撰写权/seed 编辑权/重生成权三件事捆绑变化,混淆不可修复)。代码保留但不入主实验。

### 平衡与流程

- 3 条件 × 3 个题目(开放度经机器基线预校准的短视频题材),**3×3 拉丁方**:条件顺序 × 题目-条件配对双重平衡,程序按被试计数器自动分配(取代现在的被试自选流程)。
- 每名被试:知情同意+人口学/经验筛查(4 min,操作化"零基础":自报无剧本/分镜经验 + AI 使用频率)→ 3 轮创作(每轮约 9-10 min:读题 → 写 1-2 句意图陈述 → 条件流水线 → **应用内即时问卷** 2 min)→ 结束页 + completion code(2 min)。合计 ≤40 min。
- 所有条件统一先写意图陈述再分流,保证初始输入跨条件同构。

### 被试

- **N=30**(满足 ≤30 约束):组内配对设计下,N=28 可在 α=.05、power=.80 检出 dz≈0.55(效应量参照 Doshi & Hauser 报告的相似度位移折算,正式收数前用 pilot+ghost 数据做模拟功效分析复核)。该规模符合领域先例(C&C 2024 N=36;TiiS 2024 实验一 n=30;Dramatron N=15)。
- 在线招募,**单一语言锁定**(二选一,与导师确认目标 venue 后定):Prolific 英文池(约 $8/人 × 1.33 服务费 ≈ $320-400 总额)或日本大学社群/SNS 日文招募(约 ¥800-1,000/人)。界面 i18n 已支持双语,prompts 需对应本地化。
- attention check + 最短输入校验 + 预注册排除规则;朋友 pilot 5-6 人仅用于调试,不入正式分析。

### 机器基线(全部离线生成,不占被试时间,纯 API 成本约数美元)

1. **采样地板**:每题目 × 30 份纯机器 C 式输出 → 模型固有相似度地板 + 典型度参照分布。
2. **Ghost-run 反事实**(主指标用):对每个被试 × 条件,用其真实意图陈述以完全相同的 pipeline 离线跑 **K=10 次**纯机器版本(C 式;D/D+MM 式 = 大纲零编辑直通)→ 每个人类产出都有一个配对的"如果没有人,机器会产出什么"的分布。

这直接回应该文献群公认的最大效度威胁——"同一 LLM 采样天然相似"(Padmakumar & He 只用键击溯源部分处理过)。

## 4. 指标体系(主指标全自动,人工量 ≈3 小时)

| 指标 | 计算 | 人工量 |
|------|------|--------|
| **HLZ(Human Lift z 分数,主指标)** | 格式剥离(正则去掉【景别】【时长】模板骨架,只留画面描写+台词)后,z = (d(真实终稿, ghost 质心) − ghost 内部到质心距离均值) / ghost 内部距离标准差。embedding 固定版本(主:text-embedding-3-large;稳健性:multilingual-e5)。**每被试×条件一个标量 → 可配对检验,N=30 即有功效**,摆脱"每组只有一个多样性统计量"的统计困境 | 0 |
| 集体多样性(联合主指标) | 条件×题目格内 1−平均两两余弦 + compression ratio + distinct-n(PyPI `diversity` 包,Shaib et al. 2024),控制文本长度,与机器地板并排报告 | 0 |
| 干预剂量 | D/D+MM:AI 大纲 vs 编辑后大纲的归一化 Levenshtein/token-diff(快照现有代码已保存);异议裁决按钮 + 裁决前后编辑增量 | 0 |
| 意图保真/偏离 | 意图陈述与终稿的余弦距离 × 被试逐镜头自标注(我的想法 / AI 的但我认可 / 违背本意,约 1 min)——把偏离拆成方向性的"认可扩展 vs 不认可篡改",修复原方案的方向性谬误 | 0(被试自评) |
| 心理所有权 + agency + 负荷 | 每条件提交后**应用内即时施测**:所有权 4 题(Van Dyne & Pierce 改编,α≈0.9)+ 状态 agency 2 题(Tapal SoAS,有日语版 J-SoAS)+ NASA-TLX 2 题 | 0(自报量表) |
| 质量(仅用于 H4 等价检验) | 与生成模型**不同家族**的固定 judge,4 维 rubric(连贯性/画面感/新颖性/可拍摄性),每份 3 次独立评分取均值;**30 份子样本由你+1 名同学双盲评分**验证 judge-人评相关(引用 TTCW 说明 LLM 评创意的局限,质量只主张"等价"不主张"更好") | ≈3 小时 |
| 时间/行为 | 分阶段净创作时间(**剔除 LLM 流式等待**)、重生成次数、各事件时间戳 | 0 |

## 5. 分析计划(预注册后执行)

1. **主分析**:线性混合模型 `DV ~ 条件 + 题目 + 顺序位置 + (1|被试)`(DV = HLZ、所有权、agency、质量、净时间),计划对比 D−C 与 D+MM−D,Holm 校正;Wilcoxon 符号秩作稳健性检验。
2. **集体多样性**:格内两两相似度非独立 → Mantel 型置换检验(10,000 次重排条件标签)+ bootstrap CI,与机器地板 CI 并排——多样性主张必须超出采样地板。
3. **等价检验**:质量与时间的 D vs C、D+MM vs D 用 TOST(±0.4 dz),让"无差异"成为可解释结论。
4. **剂量-反应**:HLZ/所有权 ~ 编辑距离 + 裁决类别(被试内中心化混合模型);探索性中介:编辑量 → ghost 偏移 → 所有权。
5. **敏感性分析**(全部预注册):更换 embedding 模型、含/不含格式剥离、含/不含零编辑被试(intention-to-treat 为主)、含时长协变量。
6. 分析代码在收数**前**于 pilot 数据上全部跑通并随 OSF 预注册冻结。

## 10. 风险与对策

| 风险 | 对策 |
|------|------|
| D 条件零编辑率高(操纵失败) | 界面提示"请至少调整大纲使其符合你的想法";pilot 若编辑率 <30%,加最小停留时间或逐条大纲确认;剂量-反应分析把低编辑变异转化为信息 |

| 组内残留效应 | 拉丁方全平衡 + LMM 顺序项;顺序×条件交互显著时做"仅第一节"组间敏感性分析 |

| API 模型中途漂移/下线 | 锁定版本快照;所有生成集中在 2-3 周窗口;参数逐条入库;预注册备用模型替换规则 |

## 11. 论文贡献声明(预演)

1. **实证贡献**:首个在多阶段创作流水线内操纵"人类干预位置 + AI 主动异议",并同时测量产出同质化、意图保真与心理所有权的对照实验;填补"自动化梯度 × 同质化 × 所有权"联合映射的空白。
2. **机制贡献**:ModeMirror——以模型自身众数输出为锚点的建设性异议机制,把 mode collapse 文献武器化为干预手段(从 Sarkar/Antagonistic AI 的立场文到首个可运行系统 + RCT)。
3. **方法贡献**:ghost-run 反事实分布 + HLZ 被试级配对指标,同时解决"LLM 采样天然相似"混淆与小样本功效两个公认痛点,可被后续研究复用。
4. **任务域贡献**:零基础用户 × 短视频分镜脚本的首个对照实验(区别于 Dramatron 的专业编剧、ReelFramer 的无对照系统)。
