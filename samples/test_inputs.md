# 测试用输入清单(模拟被试)

> 手动测试时复制粘贴用。更快的方式:侧边栏「实验员工具」解锁(默认密码 `nova`)后,
> 有「跳过同意+筛查」和「一键填充当前页输入」两个按钮,基本不用打字。
> 全自动回归:`.venv/bin/python scripts/dev_smoke_e2e.py`(LLM 打桩,不耗 API)。

## 0. 背景问卷(如果不用跳过按钮)

全选第一个选项即可;自评选 1;两道小测验都选「我不知道」。

## 1. 故事创意(intent,每题两个备选)

**期末周渡劫**
- 主角发现卷子上的题目自己昨晚全都梦到过,但怎么也想不起答案
- 主角和室友约好互相抽查,结果两人轮流睡着,梦里还在背书

**错过的末班车**
- 主角错过末班车,索性跟着一支深夜跑团一路跑回了家
- 主角在空无一人的站台捡到一张写着"等我"的旧车票

**五分钟的告白**
- 主角把要说的话写在毕业帽内侧,结果帽子被风吹走了
- 主角排练了一百次的告白,被对方一句"我知道"提前剧透

## 2. 大纲编辑(D / E 条件,粘贴到大纲末尾)

- `(我的修改:结局反转——主角把这一切拍成视频发到了网上,火了)`
- `(我的修改:把第2镜改成主角的宠物猫视角)`
- `(我的修改:删掉旁白,全程只用环境音)`

## 3. ModeMirror 裁决(E 条件)

| 裁决 | 理由示例 |
|------|----------|
| 采纳 | 反提案确实比我的更有记忆点 |
| 部分采纳 | 反转可以,但保留温情基调 |
| 拒绝 | 我就想要平淡真实的感觉,不要戏剧化 |

## 4. 每轮问卷

所有权 3 题选 5;主导感 2 题选 4;负荷选 3;违背本意选 2;想象匹配选 6;
**第 2 轮注意力检查选 2**;逐镜头标注全选「主要是我的想法」。

## 5. 实验员验证点

- 数据后台(researcher 模式):participants/trials/events/questionnaires 四表行数 = 1 / 3 / ≥30 / 3
- trials 表:condition 列应为该被试拉丁方序列的顺序;E 行的 dissent_json/adjudication 非空
- 完成页出现 8 位完成码
