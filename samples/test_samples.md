## 样本 1 · Flow 1（ABC，全自动一键生成）

### 侧边栏
- **受试者编号**：`U_TEST_001`
- **实验分流**：流程一：A → B → C

### A 组 · 纯人工

**分镜脚本**：
```
镜头1（5秒，全景）：
凌晨 0:47，宿舍床头小台灯亮着。主角靠在床头，腿上摊开厚厚的专业书"统计学第一章"。
画面：手机屏幕亮起一条新通知"距离考试还有 8 小时 13 分"。
台词/音效：闹钟滴答声 + 主角倒吸一口凉气："就……再翻一页。"

镜头2（5秒，特写）：
书页特写。主角拿马克笔疯狂划重点，但每一句话都被划上了。
画面：从精致的彩色高亮，逐渐变成整页全部涂黑。镜头拉远，桌面摆着三杯空咖啡。
台词/音效：马克笔尖摩擦纸张的"嗤嗤"声，咖啡杯被碰倒的咣当声。

镜头3（5秒，俯拍）：
凌晨 4:02。主角脸朝下趴在书本上睡着了，嘴角还挂着马克笔的墨。
画面：窗外天色微亮，闹钟开始震动，但主角毫无反应。
台词/音效：闹钟铃声渐强（"叮叮叮"），叠化为远处教学楼的上课铃。
```

**核心创意梗**（一句话）：
```
学渣通宵备考，把整本书都涂黑了，最后趴在书上睡死过去
```

→ 点击【提交 A 组数据】

### B 组 · 低自动化

**核心创意梗**（系统已自动从 A 带过来，可以保持或微调）：
```
学渣通宵备考，把整本书都涂黑了，最后趴在书上睡死过去
```

**给 AI 的补充提示词**：
```
偏向荒诞喜剧风格，多用夸张表情和肢体语言，少对白多内心独白；
镜头节奏要快，第三镜要有反差感（前两镜紧张，第三镜彻底躺平）。
```

→ 点击【AI 生成脚本】→ 看效果，可以再点【重新生成】覆盖一版 → 点击【提交 B 组数据】

### C 组 · 全自动一键生成

**核心创意梗**：已锁定显示，不可编辑。

→ 直接点击【一键生成】→ 点击【提交 C 组数据】 → 进入完成页

## 样本 2 · Flow 2（ABD，半路大纲干预）

### 侧边栏
- **受试者编号**：`U_TEST_002`
- **实验分流**：流程二：A → B → D

> 提示：测完样本 1 后，点完成页的「为下一位受试者重置」即可清空进度。

### A 组 · 纯人工

**分镜脚本**：
```
镜头1（5秒，中景）：
晚上 11:30 自习室。主角坐在最后一排，电脑屏幕开着 B站，耳机里在听"高数三小时速通"。
画面：屏幕角落浮窗——明早 8 点专业课考试倒计时。主角嘴里咬着一支没拆封的笔。
台词/音效：B站讲解声 + 周围键盘的哒哒声，主角嘴里小声："来得及来得及。"

镜头2（5秒，侧拍）：
快进感。主角桌面从整洁变成废墟：能量饮料、外卖盒、卷成团的草稿纸。
画面：墙上电子钟时间从 11:30 跳到 03:15。主角头发越来越乱，眼神越来越空洞。
台词/音效：键盘敲击+笔尖断裂的"啪"声，BGM 是逐渐扭曲的电子钢琴。

镜头3（5秒，全景）：
清晨 6:00。主角靠在椅子上睡着了，手里还捏着没翻完的最后一页讲义。
画面：自习室空无一人，朝阳从窗户照进来，主角脸上有键盘印的"WASD"。
台词/音效：寂静，只剩主角均匀的呼吸声，远处麻雀叫。
```

**核心创意梗**（一句话）：
```
自习室通宵的最后一夜，从信心满满到彻底躺平的两小时崩溃实录
```

→ 点击【提交 A 组数据】

### B 组 · 低自动化

**核心创意梗**（默认沿用 A，可微调）：
```
自习室通宵的最后一夜，从信心满满到彻底躺平的两小时崩溃实录
```

**给 AI 的补充提示词**：
```
节奏紧张，类似纪录片访谈风；
镜头多用主观视角和环境特写，少正脸；
台词偏意识流自言自语，体现疲惫到放弃的心理变化。
```

→ 点击【AI 生成脚本】→ 满意后点击【提交 B 组数据】

### D 组 · 半路大纲干预

**核心创意梗**：已锁定显示。

**步骤 1**：点击【生成中间大纲】→ AI 输出 3 句故事大纲。

**步骤 2**：在大纲文本框里改成（这是"干预"的关键，体现个性化）：
```
第一句：主角刚坐进自习室时还在朋友圈发"今晚我必通宵搞定"，结果先刷了 40 分钟手机。
第二句：凌晨三点开始进入幻觉模式，桌上的咖啡杯都开始有表情，公式像蚂蚁在爬。
第三句：清晨闹钟响时主角已经在草稿纸上写满"我是谁我在哪"，最后冲进考场带错了书。
```

**步骤 3**：点击【确认并生成最终脚本】→ 检查 AI 是否真的按你改后的大纲走（带错书的细节有没有出现）→ 点击【提交 D 组数据】 → 完成。
